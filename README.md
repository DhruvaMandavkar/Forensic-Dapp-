## рџљЂ Step-by-Step Setup

### Step 1 вЂ” Supabase Setup

1. Go to [supabase.com](https://supabase.com) and create a new project
2. In **SQL Editor**, paste and run the entire contents of `backend/schema.sql`
3. Go to **Authentication > Settings** and enable Email auth
4. Go to **Project Settings > API** and copy:
   - `Project URL` в†’ `VITE_SUPABASE_URL`
   - `anon public` key в†’ `VITE_SUPABASE_ANON_KEY`

### Step 2 вЂ” Compile the Smart Contract

Option A вЂ” Online (recommended for beginners):
1. Go to [remix.ethereum.org](https://remix.ethereum.org)
2. Create a new file, paste `smart-contract/ForensicEvidenceRegistry.sol`
3. In the **Solidity Compiler** tab, select version `0.8.19`, click **Compile**
4. Copy the ABI and Bytecode for use in the deploy script

Option B вЂ” CLI:
```bash
npm install -g solc
solcjs --bin --abi smart-contract/ForensicEvidenceRegistry.sol --output-dir smart-contract/out
```

### Step 3 вЂ” Deploy to Polygon Amoy Testnet

1. Get test MATIC from [faucet.polygon.technology](https://faucet.polygon.technology)
2. Install deploy dependencies:
```bash
cd forensic-dapp
npm install ethers dotenv
```
3. Create `.env` in root:
```
PRIVATE_KEY=your-metamask-private-key
RPC_URL=https://rpc-amoy.polygon.technology
CONTRACT_BYTECODE=0x...  (from compilation step)
```
4. Run the deploy script:
```bash
node scripts/deploy.js
```
5. The script will print the contract address and write `contractInfo.json` to `frontend/src/services/`

**Alternative: Deploy via Remix**
1. In Remix, go to **Deploy & Run Transactions**
2. Environment в†’ Injected Provider (MetaMask)
3. Network: Polygon Amoy (Chain ID 80002)
4. Click **Deploy**
5. Copy the deployed contract address

### Step 4 вЂ” Frontend Setup

```bash
cd frontend

# Copy env file
cp .env.example .env

# Edit .env with your values:
# VITE_SUPABASE_URL=https://your-project.supabase.co
# VITE_SUPABASE_ANON_KEY=your-anon-key
# VITE_CONTRACT_ADDRESS=0xYourDeployedAddress

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Step 5 вЂ” MetaMask Setup

1. Install [MetaMask](https://metamask.io) browser extension
2. Add Polygon Amoy network:
   - Network Name: `Polygon Amoy`
   - RPC URL: `https://rpc-amoy.polygon.technology`
   - Chain ID: `80002`
   - Symbol: `MATIC`
   - Explorer: `https://www.oklink.com/amoy`
3. Import a wallet with test MATIC

## в›“пёЏ Blockchain Integration

The smart contract records:
- **Evidence** вЂ” hash, description, confidentiality flag
- **Chain of Custody** вЂ” every handler, action, timestamp
- **Evidence Verification** вЂ” on-chain verified flag

All reads are gas-free (view functions). Writes require:
- Wallet connected to Polygon Amoy
- User's wallet assigned `ForensicOfficial` role on-chain by contract owner

**Role assignment on-chain:**
After deploying, the contract owner can assign roles:
```javascript
// In browser console or Remix
contract.assignRole("0xUserWalletAddress", 1) // 1 = ForensicOfficial
```

## рџ”’ Security Features

- **Row Level Security (RLS)**: All Supabase tables have RLS policies
- **Role-based routing**: ProtectedRoute blocks unauthorized access
- **MetaMask gate**: Forensic officials can't access dashboard without wallet
- **Email validation**: Students must use institutional email
- **Blockchain immutability**: Evidence hashes stored on-chain
- **Chain of custody**: Every transfer is logged on-chain + in DB
